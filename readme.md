Install
========================
1 - Clone : git clone https://github.com/molodoi/LaravelRestApiHateoas.git

2 - Make : composer install

3 - Create database name restfullapi

4 - Make : php artisan migrate 

5 - Make : php artisan db:seed

6 - Make : npm install
